# CareLink

Desktop Java application for managing requests and complaints from students with disabilities within a university.

## Project Structure

- `src/model/` → data models
- `src/view/` → graphical interfaces
- `src/controller/` → controllers
- `src/dao/` → database access classes
- `src/utils/` → utility classes

## Technologies
- Java
- JavaFX or Swing
- MySQL
- JDBC
- MVC Architecture
